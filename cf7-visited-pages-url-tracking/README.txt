== Description ==

Add visited site links to the mail of contact form 7, So admin/user can track that the user had which page visited before submit the form.

### Features And Options:
* Track visited pages before submit contact form 7


== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the "CF7 Visited Pages URL Tracking" plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= How to setup Plugin? =

Does not required much setup. Just activate the plugin and add [CF7VPUT_VISITED_Details] to the mail body.
